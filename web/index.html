Mission

Create a comprehensive, community-driven database of Linux hardware compatibility information to help users make informed decisions about their hardware choices.

All data is anonymized and privacy-preserving. No personally identifiable information is collected or stored in the database.

Open Source

This project is completely open source under the AGPL-3.0 license. Data is available under CC0 - no rights reserved.

Community Driven

Hardware compatibility reports are contributed by the Linux community. Everyone can contribute data and help improve hardware support.

Contribute Data

Help make Linux hardware compatibility better for everyone by contributing your hardware reports.

1

Install the Tool

cargo install lx-hw-detect
2

Generate Report

lx-hw-detect --privacy-level medium --output my-hardware.json
3

Submit via GitHub

Create a pull request with your report in the hardware-reports/ directory.
